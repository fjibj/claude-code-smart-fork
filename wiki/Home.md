## 📖 简介

Smart Fork 是一个为 Claude Code 设计的智能会话分叉工具，它使用 AI 驱动的语义搜索来帮助你找到并恢复相关的历史会话。

### 核心功能

1. **语义搜索** - 基于含义而非关键词匹配
2. **跨项目搜索** - 在所有项目中查找会话
3. **本地存储** - 数据完全存储在本地，保护隐私
4. **Slash 命令集成** - 与 Claude Code 无缝集成
